# 🚀 Mars 2020 Contributor Badge - SVG Assets

## About the Badge

The **Mars 2020 Contributor** is a historical, no-longer-available achievement badge.

### Badge Overview

- **Name:** Mars 2020 Contributor
- **Year:** 2021
- **Status:** 🚫 **UNAVAILABLE** (Historical artifact)
- **Achievement Type:** *Limited Event* 🏛️

## Historical Information

This badge was awarded to celebrate contributions to the code that runs on NASA's Mars 2020 rover (Perseverance).

### Requirements (No Longer Available)

- Code contributions to NASA/JPL repositories
- Mars 2020 mission GitHub projects
- Time-limited availability (2021)

## About Historical Badges

GitHub occasionally releases limited-time badges for:

- **Special events:** Mars 2020 mission, etc.
- **One-time initiatives:** Anniversary celebrations
- **Partnership programs:** NASA, historical partners

Once the event ends, the badge becomes **historically unobtainable**.
