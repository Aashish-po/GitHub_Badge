# 💝 GitHub Sponsor Badge - SVG Assets

## About the Badge

The **GitHub Sponsor** badge marks users who support open-source maintainers financially.

### Badge Overview

- **Name:** GitHub Sponsor
- **Requirement:** Sponsor any amount ($1+)
- **Variants:** Standard badge only
- **Achievement Type:** *Easy* ⭐⭐
- **Support Model:** Monthly recurring or one-time

## How to Earn

### Quick Steps:

1. Visit a maintainer's GitHub Sponsors page
2. Choose a sponsorship tier
3. Complete payment ($1-$1,000+/month)
4. Badge appears on your profile

### Finding Sponsors Pages

```bash
# On any GitHub profile, look for:
❤️ Sponsor (button) → View GitHub Sponsors

# Or visit directly:
https://github.com/sponsors/USERNAME
```

## Sponsorship Tiers

Most maintainers offer:

| Tier | Cost | Benefit |
|------|------|----------|
| Supporter | $1-5/month | Public sponsor recognition |
| Contributor | $10-25/month | Direct supporter badge |
| Hero/Champion | $50+/month | Premium benefits |

## Why Sponsor?

✨ **Benefits:**
- Support open-source directly
- Developer gets recurring income
- Your name appears on profile
- Often includes perks (Discord, priority PRs, etc.)
- Ethical and sustainable
