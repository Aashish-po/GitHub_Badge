# 🦈 Pull Shark Badge - SVG Assets

## About the Badge

The **Pull Shark** badge recognizes your contribution to open-source with merged pull requests.

### Badge Overview

- **Name:** Pull Shark
- **Variants:** Bronze (4 PRs), Silver (16 PRs), Gold (128 PRs)
- **Achievement Type:** *Medium* ⭐⭐
- **Progression:** Cumulative across all repositories

## Badge Progression

| Tier | PR Count | Achievement |
|------|----------|-------------|
| 🥉 Bronze | 4 PRs merged | Pull Shark Badge (Bronze) |
| 🥈 Silver | 16 PRs merged | Pull Shark Badge (Silver) |
| 🥇 Gold | 128 PRs merged | Pull Shark Badge (Gold) |

## How to Earn

### Genius Method (Recommended)

1. Find beginner-friendly open-source projects
2. Fix bugs, improve documentation, add tests
3. Submit quality pull requests
4. Collaborate with maintainers
5. Get PRs merged

**Timeline:** 1-12+ months depending on tier
