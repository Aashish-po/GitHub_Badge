# 💬 Pair Extraordinaire Badge - SVG Assets

## About the Badge

The **Pair Extraordinaire** badge recognizes co-authored commits in merged pull requests.

### Badge Overview

- **Name:** Pair Extraordinaire
- **Variants:** Bronze (1), Silver (10), Gold (24)
- **Achievement Type:** *Medium* ⭐⭐⭐
- **Focus:** Collaboration and pair programming

## Badge Progression

| Tier | Co-authored Commits | Achievement |
|------|---------------------|-------------|
| 🥉 Bronze | 1 commit | Pair Extraordinaire (Bronze) |
| 🥈 Silver | 10 commits | Pair Extraordinaire (Silver) |
| 🥇 Gold | 24 commits | Pair Extraordinaire (Gold) |

## How to Earn

### Genius Method (Recommended)

1. **Find collaborators:**
   - Team members on your project
   - Open-source contributors
   - Friends interested in coding

2. **Pair program together:**
   - Work on same code simultaneously
   - Share keyboard and screen
   - Discuss and decide on approach

3. **Add co-author to commits:**

```bash
git commit -m "Implement feature X

Co-authored-by: Friend Name <friend@example.com>
Co-authored-by: Another Dev <another@example.com>"
```

4. **Merge the PR:**
   - Get PR reviewed and approved
   - Merge to main branch

**Timeline:**
- Bronze: 1 week
- Silver: 2-4 months
- Gold: 6+ months

## Best Practices

✅ **Authentic Collaboration:**
- Both people contribute meaningfully
- Use with real development work
- Valuable for team learning
- Builds relationships

❌ **Avoid:**
- Fake co-authors
- Adding names without contribution
- Exploiting the system
