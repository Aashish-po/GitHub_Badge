# ⚡ Quick Draw Badge - SVG Assets

## About the Badge

The **Quick Draw** badge rewards merging pull requests within 5 minutes of opening them.

### Badge Overview

- **Name:** Quick Draw
- **Requirement:** Merge 1 PR within 5 minutes
- **Variants:** Standard badge only
- **Achievement Type:** *Easy* ⭐
- **Speed:** Lightning-fast! ⚡

## How to Earn

### Simple Steps:

1. Create a pull request
2. Merge it within 5 minutes
3. Badge earned!

### Timeline
Instant - Can be done in your first day on GitHub!

## Real-World Use Case

**Fixing an urgent bug:**

```bash
# Branch for quick fix
git checkout -b hotfix/critical-issue

# Make your fix
echo "# Fixed critical bug" >> CHANGES.md

# Commit and push
git commit -am "Fix: Critical security issue"
git push origin hotfix/critical-issue

# Create PR
gh pr create --title "Fix critical issue"

# Review and merge immediately (within 5 min!)
gh pr merge <number> --merge

# 🎉 Badge earned!
```
