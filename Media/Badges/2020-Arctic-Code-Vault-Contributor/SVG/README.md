# 🏔️ Arctic Code Vault Contributor Badge - SVG Assets

## About the Badge

The **Arctic Code Vault Contributor** is a historical, no-longer-available achievement badge.

### Badge Overview

- **Name:** Arctic Code Vault Contributor
- **Year:** 2020
- **Status:** 🚫 **UNAVAILABLE** (Historical artifact)
- **Achievement Type:** *Limited Event* 🏛️

## Historical Information

This badge was awarded to celebrate contributions to significant open-source repositories that were preserved in the Arctic World Archive.

### About the Arctic Code Vault

GitHub participated in a historic digital preservation initiative:

- **Purpose:** Preserve open-source code for future generations
- **Location:** Svalbard, Norway (Arctic)
- **Format:** Microfilm stored in permafrost
- **Projects:** Most active open-source repositories on GitHub as of February 2020
- **Permanence:** Designed to last 1,000+ years

## Requirements (No Longer Available)

To earn this badge (in 2020):
- Have a public repository on GitHub
- Be active before the snapshot date (Feb 2, 2020)
- Contribute to preserved projects automatically
