# 🦄 YOLO Badge - SVG Assets

## About the Badge

The **YOLO** (You Only Live Once) badge is earned by merging a pull request without review in a repository.

### Badge Overview

- **Name:** YOLO
- **Requirement:** Merge 1 PR without review (in your own repository)
- **Variants:** Standard badge only
- **Achievement Type:** *Easy* ⭐

## How to Earn

1. Create a pull request in your own repository
2. Merge it without waiting for reviews
3. Badge automatically granted!

## SVG Files Included

This folder contains the SVG version of the YOLO badge.

### Available Formats

- **SVG Format** (Scalable Vector Graphics) - For web and high-quality printing
- See `PNG/` folder for PNG raster versions
