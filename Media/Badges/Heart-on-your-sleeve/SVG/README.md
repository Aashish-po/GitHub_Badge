# ❤️ Heart on Your Sleeve Badge - SVG Assets

## About the Badge

The **Heart on Your Sleeve** badge celebrates users who create repositories for Valentine's Day.

### Badge Overview

- **Name:** Heart on Your Sleeve
- **Requirement:** Unreleased/limited availability
- **Achievement Type:** *Special Event* ⭐
- **Occasion:** Valentine's Day themed

## SVG Files Included

This folder contains the SVG version of the Heart on Your Sleeve badge.

### Available Formats

- **SVG Format** (Scalable Vector Graphics)
- **PNG Format** - See `PNG/` folder

## Usage

```html
<img src="HeartOnYourSleeve.svg" alt="Heart on Your Sleeve" width="100">
```

## About Special Event Badges

GitHub occasionally releases limited-time badges for special occasions:

- **Arctic Code Vault Contributor** (2020) - Historical
- **Mars 2020 Contributor** (2021) - Historical  
- **Heart on Your Sleeve** (Valentine's Day) - Limited

These badges are not permanently available and require specific conditions.
