# 💻 GitHub Pro Highlight - SVG Assets

## About the Highlight

The **GitHub Pro** highlight is a premium badge that showcases your GitHub Pro subscription status.

### Highlight Overview

- **Name:** GitHub Pro
- **Type:** Subscription Badge
- **Status:** Active/Available
- **Renewal:** Monthly or annual subscription
- **Mode Variants:** Light Mode and Dark Mode

## What is GitHub Pro?

GitHub Pro is a paid subscription that unlocks premium features:

### Features Included

✨ **GitHub Pro Benefits:**
- Advanced code review tools
- Scheduled reminders
- Advanced filtering in issues and pull requests
- GitHub Pages with custom domain support
- Code owners
- Auto-linked references
- Sponsorships insights
- Profile README
- And more...

## How to Get

### Steps to Enable:

1. Go to [github.com/settings/billing](https://github.com/settings/billing)
2. Click "Upgrade to GitHub Pro"
3. Choose monthly or annual plan
4. Complete payment
5. Highlight appears on your profile

### Cost

- **Monthly:** $4 USD/month
- **Annual:** $40 USD/year (saves $8)
- **Free Trial:** 1 month free
